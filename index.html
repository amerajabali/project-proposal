Application Overview

MathWizz is a fun and interactive web application that allows users to test their arithmetic skills through a series of randomly generated math problems. Users can select from operations like addition, subtraction, multiplication, and division. Each response provides instant feedback and tracks the user's score in real-time.

Technology Overview

  • Languages: HTML, CSS, JavaScript, JSX
  • Libraries: React.js, Tailwind CSS, Framer Motion
  • Platforms: Vite, Node.js
  • Hosting: Vercel / Netlify
